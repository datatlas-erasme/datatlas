# Datatlas

## Development

Run `prettier` on whole repository:

```
nx format:write
```

Run `lint` on whole repository

```
nx run-many --target=lint
```

### Frontend

```
nx serve frontend
```

### Backend

```
nx serve backend
```

> Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.
